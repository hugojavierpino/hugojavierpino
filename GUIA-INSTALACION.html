Guía de configuración

hugojavierpino.org · Panel de administración con Netlify CMS

Esta guía te explica paso a paso cómo conectar GitHub con Netlify para poder publicar artículos, sermones, videos y descargas desde un panel visual, sin tocar código.

Tiempo estimado: 20–30 minutos la primera vez. Después, publicar contenido nuevo tarda menos de 2 minutos.

Parte 1 — Crear tu cuenta en GitHub

1

Registrarte en GitHub

Andá a github.com y creá una cuenta gratuita. Elegí un nombre de usuario simple, por ejemplo hugopino o hjpino.

2

Crear un repositorio nuevo

Una vez dentro de GitHub, hacé clic en el botón verde "New" o "+" en la esquina superior derecha → New repository.

  • Nombre del repositorio: hugojavierpino
  • Visibilidad: Public (necesario para Netlify CMS gratuito)
  • Marcá la opción "Add a README file"
  • Clic en Create repository
3

Subir los archivos del sitio

En tu repositorio recién creado, hacé clic en "Add file" → "Upload files". Descomprimí el ZIP del sitio y arrastrá todos los archivos y carpetas a esa pantalla. Después hacé clic en "Commit changes".

4

Editar el archivo de configuración del CMS

En tu repositorio, andá a la carpeta admin → hacé clic en config.yml → clic en el ícono de lápiz (editar). Buscá esta línea:

repo: TU-USUARIO-GITHUB/hugojavierpino

Reemplazá TU-USUARIO-GITHUB por tu nombre de usuario real de GitHub. Guardá haciendo clic en "Commit changes".

Parte 2 — Conectar Netlify con GitHub

5

Importar el sitio desde GitHub

En Netlify, en lugar de arrastrar la carpeta como antes, andá a Sites → Add new site → Import an existing project. Elegí GitHub y autorizá el acceso. Seleccioná el repositorio hugojavierpino.

En la configuración del deploy, dejá todo como está y hacé clic en Deploy site. En 1–2 minutos el sitio va a estar publicado.

6

Agregar tu dominio

En Netlify → Domain settings → Add custom domain → escribí hugojavierpino.org. Netlify te va a dar los nameservers para configurar donde compraste el dominio.

Parte 3 — Activar el panel de administración

7

Habilitar Netlify Identity

En Netlify → tu sitio → pestaña Identity → clic en "Enable Identity". Esto crea tu sistema de login para el panel.

8

Habilitar Git Gateway

Dentro de Identity → Services → Enable Git Gateway. Esto conecta el panel con tu repositorio de GitHub.

9

Crear tu usuario de administrador

Todavía en Identity → Invite users → ingresá tu email personal. Te va a llegar un correo de invitación. Hacé clic en el link, creá tu contraseña, y listo.

10

¡Entrar al panel!

Andá a hugojavierpino.org/admin. Vas a ver el panel de administración. Desde ahí podés:

  • Escribir y publicar artículos del blog
  • Agregar sermones con sus archivos de audio
  • Cargar videos con el link de YouTube
  • Subir recursos para descarga

Cómo publicar contenido nuevo

Publicar un artículo del blog

Panel → Blog → New Artículo → completás título, categoría, fecha, resumen y cuerpo del texto → Publish. En 30 segundos aparece en el sitio.

Subir un sermón en audio

Panel → Audios / Sermones → New Sermón → completás el título, serie, pasaje bíblico → subís el MP3 o pegás el link de SoundCloud/Spotify → Publish.

Agregar un video

Panel → Videos → New Video → pegás el link de YouTube, completás el tipo y descripción → Publish.

Subir un recurso descargable

Panel → Descargas → New Recurso → subís el PDF/DOCX/MP3 y completás el título → Publish.

Tip: Para los audios, podés usar SoundCloud o Anchor (Spotify for Podcasters) de forma gratuita y pegar el link en el campo correspondiente. Así no usás el espacio de Netlify para archivos pesados.

Resumen de servicios utilizados

  • GitHub — gratuito — guarda los archivos del sitio
  • Netlify — gratuito (plan Starter) — publica el sitio y da el panel CMS
  • Netlify CMS — gratuito — el panel visual de administración
  • Dominio hugojavierpino.org — el que ya tenés
